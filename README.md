>

### **介绍**
>My-Python-Journey 的初衷是为了完成我这几年来学习 python 的心路历程，从一个初学者到测试工程师，到自动化测试工程师，最后到测试开发工程师。
> 
>我是如何一步一步实现的。
> 
> 也希望通过自己的经验可以帮助更多的初学者，少走弯路。
> 

通过这个 repo，你可以学习到：
>1. python 基础语法
>2. python 面向对象
>3. python 进阶语法
>4. python 常用库
>5. python 自动化测试
>6. python 测试开发
>7. python 性能测试
>8. python 爬虫
>9. 测试开发扩展

给喜欢我的 repo 的你一些建议：

- 坚持阅读，坚持学习。（keep reading, keep learning）
- 多动手，多实践。（more hands-on, more practical)
